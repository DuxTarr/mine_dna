# Mine PVP

### Genome specification

##### Appearance genome:

> 0x EE MM HE OW JJ C X XX XX \
> EE - eyes;\
> MM - mouth;\
> HE - hair and ears;\
> OW - outerwear;\
> PP - pants;\
> JJ - jacket;\
> C - skin color;\
> X XX XX - special items;

##### Skills genome

> 0x H D P X XX XX\
> H - health;\
> D - damage;\
> P - protection;\
> X XX XX - special skills\
